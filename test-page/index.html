1 · Install the connector

Download and double-click. Per-user install, no admin prompt. Tray icon appears (lime "L" square, bottom-right).

↓ Download LekhaTallyAgentSetup.exe

2 · Accept the self-signed cert (one time)

The connector uses HTTPS with a self-signed cert. Your browser blocks calls until you accept it once:

  1. Open https://127.0.0.1:9100/health in a new tab.
  2. Click Advanced → Proceed to 127.0.0.1 (unsafe).
  3. You should see {"ok":true,...}. Close the tab.

3 · Paste pairing token

Right-click tray icon → Show pairing token → Notepad opens with a UUID → copy and paste below.

Stored only in your browser's localStorage.

Select a template or write your own XML. The connector forwards it as-is to Tally on localhost:9000.
