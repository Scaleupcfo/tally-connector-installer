Lekha Tally Agent — Test Page

Validates the local agent against your Tally. Once verified, this flow gets baked into Lekha AI.

1 · Install the agent

Download and double-click. Per-user install, no admin prompt. Tray icon appears (blue square, bottom-right).

↓ Download LekhaTallyAgentSetup.exe (3.84 MB)

2 · Accept the agent's self-signed cert (one time)

The agent uses HTTPS with a self-signed cert. Your browser will block calls to it until you accept the cert once:

  1. Open https://127.0.0.1:9100/health in a new tab.
  2. You'll see "Your connection is not private." Click Advanced → Proceed to 127.0.0.1 (unsafe).
  3. You should see {"ok":true,...}. Close the tab.

3 · Paste pairing token

Right-click tray icon → Show pairing token → Notepad opens with a UUID → copy and paste below.

Stored only in your browser's localStorage.

4 · List loaded companies

Returns every company currently open in Tally Prime, with each one's books period.

5 · List ledgers

Master ledger list (name, parent group, opening/closing balance, GSTIN) for one company.

6 · List vouchers

Voucher headers + ledger entries in a date range. Can take 30 seconds on big companies.
